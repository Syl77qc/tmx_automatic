📖 Méthodologie

Source des signaux — 56 paires cause→effet issues du test de causalité de Granger (lags 1–5 jours, p<0.05) sur 25 ans de données EOD, filtrées sur WR>55%, Sharpe>0.5, N≥20.


Condition d'activation — Le signal est ACTIF si le rendement cumulé de la cause sur lag jours est inférieur à −1σ×√lag (σ = écart-type journalier sur 252 jours).


Confirmation — Le signal est CONFIRMÉ si le z-score 20 jours de l'effet est < −1.5, indiquant une condition de survente indépendante.


Limites — Les signaux Granger mesurent la prédictibilité linéaire passée, non la causalité économique. Ce site est un projet de recherche personnel. Ce ne sont pas des conseils financiers.
