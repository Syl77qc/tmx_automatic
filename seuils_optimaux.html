Cette page documente l'optimisation des seuils par rendement J+20, utilisée dans le PRD v2.2. Les seuils opérationnels du système actuel ont été fixés par test de Wilcoxon sur 25 ans — tous les FNBs actifs utilisent 2,0σ. Les FNBs marqués CTX sont des indicateurs contextuels en v3.0 : leurs chocs alimentent les signaux de contagion mais aucune position mean reversion n'est ouverte sur eux. → Voir l'univers actuel sur la page Système
🎯

Si un FNB continue systématiquement de baisser après le signal de 2σ, c'est que le vrai bas statistique est plus bas. Relever le seuil permet d'entrer plus près du creux réel, réduisant le drawdown post-entrée et améliorant le rendement net. Cette analyse teste cette hypothèse pour chaque FNB sur 25 ans.

Robustesse — Impact des crises
Variation du rendement J+20 avec vs sans les périodes de crise · Stabilité sur 25 ans
Δ Rendement J+20 sans crises (GFC 2008 + COVID 2020) vs pleine période
Comment lire ce graphique. Chaque barre indique de combien le rendement J+20 change quand on retire les périodes de crise (GFC 2008 + COVID 2020). Une barre proche de 0 signifie que les crises n'ont pas influencé significativement le résultat. Tous les FNBs affichent un Δ inférieur à ±0,5% — le comportement de mean reversion est extrêmement stable dans le temps. C'est la confirmation statistique la plus importante de cette analyse : les résultats ne sont pas des artefacts des périodes exceptionnelles.
